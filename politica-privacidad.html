Finalidad

Los datos enviados mediante el formulario de contacto serán utilizados únicamente para responder consultas, elaborar presupuestos o mantener comunicaciones relacionadas con los servicios solicitados.

La base jurídica del tratamiento es el consentimiento otorgado por el usuario al enviar voluntariamente sus datos.


Conservación

Los datos se conservarán durante el tiempo necesario para atender la solicitud o mientras exista una relación comercial.


Cesión de datos

No se venderán ni cederán datos personales a terceros salvo obligación legal o cuando sea necesario para prestar el servicio solicitado.
